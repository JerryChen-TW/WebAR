# Language Lens — Google AI Edge Gallery Skill

把 **Gemma 4** 變成「拍照學語言」的視覺家教。對著任何物件拍照或上傳圖片，
它就辨識物件並教你目標語言的單字、發音和例句。
**完全離線、辨識任何物件、支援任意語言。**

---

## 為什麼是這個方案

Gemma 4 E2B / E4B **原生支援影像理解**（物件偵測、OCR、場景描述）。
所以不需要任何額外的偵測模型——Gemma 自己看圖、自己翻譯。

| | 網頁版 (MediaPipe) | Language Lens (Gemma 4 視覺) |
|--|--|--|
| 辨識種類 | 固定 80 種 | **無限（任何物件）** |
| 翻譯 | 內嵌字典 | **Gemma 即時生成** |
| 語言 | 寫死 8 種 | **任意語言** |
| 發音 / 例句 | 只有拼音 | **發音 + 例句 + 文法提示** |
| 執行 | 瀏覽器（iOS 易崩潰） | **原生 LiteRT-LM，穩定** |

---

## 安裝

### 1. 模型
Gallery → Agent Skills → 使用 **Gemma-4-E2B-it**（2.59 GB，已支援影像）。

### 2. 匯入 Skill
在手機把 `language-lens/` 這個資料夾（含 `SKILL.md`）放到可存取的位置，
在 Gallery 的 skill 匯入流程選擇該資料夾。

> 本 skill 是 **text-only（純人格）skill**，只需要 `SKILL.md`，不含 JavaScript。

---

## 使用

1. 開啟 **Gemma-4-E2B-it** 的對話，啟用 **Language Lens**
2. 告訴它要學的語言：「我想學日文」
3. 用 **Ask Image / 上傳圖片 / 拍照**，對著物件
4. Gemma 回覆：

```
🍎 蘋果
- 單字：りんご
- 發音：ringo
- 例句：りんごを食べます。（我吃蘋果。）

再給我看別的東西吧！📸
```

說「我是初學者」會用最簡單的詞；說「進階」會加同義詞和更自然的說法。
